  <name>Task 2: Create PATCH endpoint for field updates</name>
  <files>backend/app/api/review.py, backend/app/models/review.py</files>
  <action>
Create UpdateFieldRequest model:
- artikelnummer: str
- field_name: str (e.g., "bezeichnung1", "preis", "hoehe_cm")
- field_value: Any (new value for the field)

Create PATCH /api/review/{upload_id}/product endpoint:
- Accept UpdateFieldRequest in body
- Read merged_products.json
- Find product by artikelnummer
- Update field in product data object
- Set source for that field to "manual_edit" in sources object
- Write updated JSON back to file
- Return updated product
- Return 404 if upload_id or artikelnummer not found

Preserve Phase 3 wrapper structure and all other fields unchanged.
Use atomic write (write to temp file, rename) to avoid corruption.
